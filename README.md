# bmi-projekti.github.io
BMI-projektin ihan paras nettisaitti. Tän repositoryn edustama kehitysversio elelee osoitteessa [`bmi-projekti.github.io`](http://bmi-projekti.github.io), ja oikea webasaitti on osoitteessa [`bmiproject.fi`](http://bmiproject.fi).

## Kuinka devaan

Asenna ensiksi [Jekyll](http://jekyllrb.com/), se on helpointa jos käyttiksenä on Macci tai Linukka. Sitten kopioi nettisaitin Git-repository koneellesi. Tähän voit käyttää esimerkiksi [GitHubin omaa sovellusta](https://mac.github.com/). Kun repository on kopioitu, mene komentorivillä repositoryn hakemistoon ja aja siellä seuraava komento:

```
jekyll serve
```

Tämä pyöräyttää pystyyn lokaalin nettisaitin, joka elelee osoitteessa `localhost:4000`. Sit vaan tykittelet koodia ja voit ihastella kuinka muutoksesi päivittyy nettisaitille.
